Для установки и запуска программы на компьютере должна быть установленна nodejs 3+, в папке с программой нужно выполнить комманду npm install. После установки всех зависимостей заупустить локальный сервер, в phpMyAdmin импортировать базу данных db.sql из папки /db. Запустить программу коммандой npm start.
